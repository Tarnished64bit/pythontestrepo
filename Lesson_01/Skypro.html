Условия домашки

Привет, дорогой студент! Поздравляем тебя! Ты приступаешь к выполнению своего первого домашнего задания по автоматизации тестирования. Внимательно прочитай условия и соблюдай формат сдачи домашнего задания, пожалуйста. Желаем удачи)

Задание 1. Выполните упражнения

Перед выполнением заданий ознакомьтесь со стандартами оформления кода в Python и строго следуйте им.

На шаге «Подготовка к выполнению ДЗ — Настройка окружения» вы создали папку 01_lesson, именно в нее нужно будет добавить отдельные скрипты по заданиям ниже.


Создайте переменную
  1. Создайте файл lesson_1_task_1.py.
  2. В файле создайте переменную 
    my_name
    my_name
    .
  3. Присвойте переменной значение — ваше имя. Помните, что строковые значения указываются в кавычках.
  4. Напишите команду вывода значения переменной на экран. Используйте функцию 
    print
    print
    .
  5. В правом верхнем углу нажмите кнопку play. В консоль выведется значение переменной.

    6. Убедитесь, что в VS Code включено автосохранение («Файл» → «Автосохранение»).

  1. Создайте файл lesson_1_task_2.py.
  2. В файле создайте переменную

    my_age

  3. В следующей строке присвойте этой же переменной другое значение — ваш возраст через три года.
  4. Напишите команду вывода значения переменной на экран. Используйте функцию 
    print

    .
  5. Запустите скрипт.

Задание 2. Проверьте код на соответствие стилю

Основная задача - получить чистый код. Вы можете использовать любой линтер или плагин для проверки кода на соответствие стандартам PEP8.

При настройке окружения в шагах урока мы с вами установили расширение flake8. Если вы выбрали работу с этим линтером, то проверьте скрипты в папке lesson_01 на соответствие стилю, выполнив следующую команду в терминале:

flake8 lesson_01
flake8 lesson_01

Исправьте ошибки, если линтер их выявил.

Если вы не понимаете, как исправить ошибки, то задайте вопрос в обсуждении или при сдаче домашнего задания — и наставники вам помогут.

Задание 3. Сдайте домашнюю работу по формату

Создайте pull request из ветки lesson1 в master (main).

Если вдруг забыли, как это сделать, вернитесь к уроку «Системы контроля версий» или задайте вопрос в обсуждении.

Формат сдачи домашней работы

  • Ссылка на pull request из ветки lesson1 в master (main).

После того как наставник примет работу, можно будет влить ветку lesson1 в master (main).

Правильная ссылка имеет формат 

https://github.com/…/…/pull/N
https://github.com/…/…/pull/N
, где N — порядковый номер пул-реквеста.
